# bitrix-api-vuejs

### Start Server

Add user ID and webhook hash into a [config file](https://github.com/yukal/node-bitrix-lead/blob/master/server/_config.js) at first.

``` bash
$ cd server
$ npm install
$ npm run server
```

### Start Client

``` bash
$ cd client
$ npm install
$ npm run server
```

### Start With Docker

``` bash
# Start
$ docker-compose up

# Cleanup
$ docker-compose down
```
